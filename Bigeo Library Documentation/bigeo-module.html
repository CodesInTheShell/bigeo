Module bigeo

source code

Bigeo is a python based software developed for automated geospatial processing. Bigeo comes along with automated processing algorithms that can be run via command line. See documentation bigeo HOW TOs

Bigeo can also be imported as a third party library. Refer to bigeo library documentation.
